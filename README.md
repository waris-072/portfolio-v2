# 🚀 Portfolio v2

A modern, responsive, and interactive personal portfolio built with **React** to showcase my projects, technical skills, certifications, and professional journey as a **MERN Stack Developer**.

🔗 **Live Demo:** https://porfoliios.netlify.app/

---

## 📖 Overview

This portfolio highlights my experience in full-stack web development, featuring responsive UI design, reusable React components, project showcases, and a clean user experience with light and dark themes.

It serves as my professional portfolio for internships, junior developer opportunities, and freelance work.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/waris-072/portfolio-v2.git
```

### Navigate into the project

```bash
cd portfolio-v2
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
